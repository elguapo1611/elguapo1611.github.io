Intro to JS

Strings (surrounded by "")

              var s = "hello";

Add two strings together (concatenation)

              var foo = "hello";
              var bar = "world";
              alert(foo + " " + bar);

So what else can you do with a string?.... Call "methods"

Give me the first letter of var s:

              var foo = "hello"
              alert(foo.charAt(0)); // charAt is a method called on foo.

String Exercises list of string methods

              // replace livingsocial with intergalactic
              var foo = "hello there livingsocial";
            
              // convert all letters to uppercase
              var foo = "hello there livingsocial";
            
              // remove all trailing and leading whitespaces
              var foo = "    hello world    ";

Numbers (floats and integers)

              var foo = 1; // integer
              var bar = 2.3; // float

Addition, Subtraction, Multiplication, Division

              var foo = 1; // integer
              var bar = 2.3; // float
              foo + bar
              foo - bar
              foo * bar
              foo / bar

Using the Math class to manipulate numbers

              var foo = 2.3;
              alert(Math.round(foo));
